BMI Classification Android App

This is a simple Android application developed for ICS 2300: Mobile Applications Design and Development – Assignment 1.
The app calculates a user’s Body Mass Index (BMI) using their weight and height, then classifies the result.

Features

Input weight (kg) and height (m)

Calculates BMI using the standard formula

Displays BMI value and health category

Simple and user-friendly interface

BMI Formula
BMI = weight / (height × height)

BMI Categories

Underweight: BMI < 18.5

Normal: 18.5 – 24.9

Overweight: 25 – 29.9

Obese: BMI ≥ 30

Tools Used

Android Studio

Java

XML

How to Run

Open the project in Android Studio

Sync Gradle

Run on an emulator or Android device
