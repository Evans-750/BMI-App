BMI Calculator App
Introduction

The BMI Calculator App is a simple Android application designed to help users calculate their Body Mass Index (BMI) using their weight and height. The app instantly displays the BMI value along with a health classification, making it easy for users to understand their body weight status.

Understanding BMI

Body Mass Index (BMI) is a commonly used measure to assess whether a person has a healthy body weight relative to their height.

- App Features
- User Input

Allows users to enter:

Weight in kilograms (kg)

Height in meters (m)

Input fields are simple and easy to use.

 BMI Calculation

The app calculates BMI instantly when the user taps the Calculate button.

The BMI value is displayed with two decimal places for better accuracy.

 Health Classification

Based on the calculated BMI, the app classifies the user into one of the following:

Underweight

Normal

Overweight

Obese

This helps users quickly understand their health status.

 Input Validation

The app checks whether both weight and height have been entered.

If any field is left empty, the user is prompted to enter the missing information.

This prevents incorrect calculations and improves reliability.

 Clear Output Display

The result section displays:

The calculated BMI value

The corresponding BMI category

Results are shown in a clear and readable format.
